# Wanderer game

## Information

A single player, frontend, tile-based game.
The whole project is written in typescript.
For original idea and assets credit goes to: **Greenfox Academy**

Try out at: https://wanderer-bf.netlify.app/

## Controls

'W': `Move up`
'A': `Move right`
'S': `Move down`
'D': `Move left`
'SPACE': `Fight`
